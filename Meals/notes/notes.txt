üß† useLayoutEffect vs useEffect:

- useLayoutEffect runs *before* the component is painted to the screen.

‚ú® Key Points

Creates a bar at the bottom of the screen.

Each screen inside <Tab.Navigator> becomes a tab item.

Tabs automatically handle:

Screen switching animations

Safe area padding

Label + icon display (if configured)

‚úÖ 3. Adding Icons to Bottom Tabs
With @expo/vector-icons
import { Ionicons } from "@expo/vector-icons";

<Tab.Screen 
  name="Home" 
  component={HomeScreen}
  options={{
    tabBarIcon: ({ color, size }) => (
      <Ionicons name="home" color={color} size={size} />
    ),
  }}
/>

Props passed to tabBarIcon

color ‚Üí active/inactive tint color

size ‚Üí auto-adjusted size by tab bar

focused ‚Üí boolean (true when selected)

‚ùó No toggle function (like Drawer)

Bottom Tabs are always visible by default.
But you can hide them:

Hide tab bar for a specific screen
options={{ tabBarStyle: { display: 'none' } }}
